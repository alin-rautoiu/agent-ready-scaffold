# Chart Task Fit Audit

Use this when adding or revising KPI cards, charts, trend rows, or summary visuals.

## Method

1. Classify the task: trend, ranking, composition, deviation, or distribution.
2. Check whether the current visual supports that task without extra interpretation work.
3. Add thresholds, baselines, or targets where the decision needs them.
4. Remove decorative treatments that do not improve comprehension.
5. Verify on desktop and mobile.
