# Honest UX Audit

## Check

- Irreversible actions are explicit about consequences.
- Consent language is specific and not manipulative.
- AI-generated or AI-assisted content is disclosed where the user needs that context.
- Sensitive settings and credential flows avoid exposing secrets.
- Unsubscribe, invite, and public subscriber flows are transparent and reversible where appropriate.

## Lifecycle Transparency Additions

Reference sources: gov.uk/guidance/content-design/content-maintenance, gov.uk/guidance/content-design/gov-uk-content-retention-and-withdrawal-archiving-policy

- Make lifecycle states explicit (draft, sent, archived, withdrawn).
- Clarify scope and consequences for archive/withdraw actions before commit.
- Include periodic maintenance checks for stale or outdated published content.
