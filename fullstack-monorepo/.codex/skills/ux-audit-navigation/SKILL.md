# Navigation Audit

## Check

- Active page state is visually clear and announced accessibly.
- Navigation groups are meaningful and labeled.
- Mobile navigation is reachable, dismissible, and focus-safe.
- Menus are click and keyboard operable, not hover-only.
- Focus states are visible and interaction is not gated behind animation.
- Skip-link and `aria-current` behavior exist where expected.
