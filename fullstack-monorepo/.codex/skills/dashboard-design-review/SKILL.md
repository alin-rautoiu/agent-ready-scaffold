# Dashboard Design Review

Use this for dashboard pages and dense admin lists.

## Checks

1. Identify the top 1-3 user decisions on the page.
2. Verify the primary KPI appears first and supports a real decision.
3. Check that labels, units, and state context are scanable.
4. Check chart or table choice against the analytical task.
5. Validate mobile behavior and action discoverability.
6. Convert findings into implementation tasks with likely file targets.

## Output

- Findings with impact and recommended change
- Likely files to touch
- Quick verification plan
- Top-5 prioritized backlog
