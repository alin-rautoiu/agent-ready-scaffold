# Playwright CLI

Use browser automation when a change affects:

- slide readability
- responsive layout
- theme rendering
- PDF export verification

Check for:

- clipping or overflow
- weak contrast
- fragment timing issues
- uneven spacing
- regressions across `theme-neon-dusk`, `theme-natural-light`, and `theme-classical`
